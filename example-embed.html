Example Page with Embedded Chatbot

This is an example page showing how to embed the chatbot on your website.

Embedding Instructions

To embed the chatbot, add this script tag before the closing </body> tag. The CSS will be automatically loaded:

<script src="path/to/chatbot.iife.js" 
        data-api-url="https://your-api-domain.com/api/chat"
        data-title="Chat Assistant"
        data-placeholder="Type your message..."></script>

Note: Make sure both chatbot.iife.js and style.css are in the same directory.
